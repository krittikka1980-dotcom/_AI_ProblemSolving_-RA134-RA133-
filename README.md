# 🧭 Smart Navigation System (BFS vs DFS)

## 📌 Project Description
This project implements a Smart Navigation System that finds paths between nodes in a graph using Artificial Intelligence search techniques.

The system allows users to input a custom graph and compute paths between a start node and a goal node using:
- Breadth-First Search (BFS)
- Depth-First Search (DFS)

It also compares both algorithms based on performance.

---

## ✨ Features
- 🌐 Interactive web-based UI  
- 🧠 BFS and DFS algorithms implemented  
- 📊 Performance comparison:
  - Nodes explored  
  - Execution time  
- 📍 Path visualization (graph with highlighted path)  
- 🎨 Premium UI with modern design  
- ⚡ Real-time result generation  

---

## 🧠 Algorithms Used

### 1. Breadth-First Search (BFS)
- Explores level by level  
- Guarantees shortest path (in unweighted graphs)  
- Uses queue data structure  

### 2. Depth-First Search (DFS)
- Explores deeply before backtracking  
- Does not guarantee shortest path  
- Uses stack (or recursion)  

---

## ⚖️ Comparison

| Feature            | BFS                | DFS                |
|--------------------|--------------------|--------------------|
| Path Optimality    | Shortest Path      | Not guaranteed     |
| Nodes Explored     | Moderate           | May be higher      |
| Traversal Style    | Level-wise         | Depth-wise         |
| Data Structure     | Queue              | Stack              |
